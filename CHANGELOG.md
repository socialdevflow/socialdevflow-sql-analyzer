# Changelog

## [1.0.1] - 2026-06-01

### Changed
- Point plugin metadata and docs to GitHub repo: https://github.com/socialdevflow/socialdevflow-sql-analyzer
- README install instructions now use `git clone` from GitHub
- Install script shows repo URL and update command after install

## [1.0.0] - 2026-05-31

### Added
- `oracle-query-analyzer` skill: static analysis for performance issues and anti-patterns
- `oracle-index-advisor` skill: B-Tree, composite, function-based, and covering index recommendations
- `oracle-dialect-converter` skill: full MySQL/PostgreSQL/SQLite/SQL Server → Oracle 19c conversion
- `/oracle-analyze` command: full analysis workflow with index recommendations
- `/oracle-convert` command: dialect detection and conversion workflow
- `oracle-sql-best-practices` rule: auto-attached to `.sql` and Liquibase migration files
